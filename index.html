Collection of AngularJs Directives, 'ngd-*'.

Directives are very essential part in AngularJs, and we don't want to include so many javascripts page by page for each directive. This collection provides you to include only one script file ngd.min.js for full benefit of all directives;

  • ngd-code auto fixing indentation for your code
  • ngd-auto-height auto height adjustment for textarea
  • ngd-enter listener for ENTER key
  • ngd-fixed-under fixed navibation bar under a header
  • ngd-image-drop image drag and drop
  • ngd-image-paste image copy and paste
  • ngd-markdown markdown previewer
  • ngd-overlay modal blocker for any element
  • ngd-viewport navigation tracking for current viewport
  • ngd-form-unsaved warning message when form not saved
  • ngd-tree Deep-level collapsale list presentation

To use this in your app, simply include javascript 

            <script src="ngd.js"></script>
optionally, for your app, include ngd module 

            var myApp = angular.module('myApp', ['ngd']);
NOTE: This page only uses "ngd" directive with no app, controller, nor jQuery.

ngd-code Download Source

We want to type code and show to the users. However, <pre> tags cares every single space and does not care about your indentation for your html and ends up showing ugly. If you want to make it look pretty to the user, your html indentation must be ignored.

"ng-code" does read your html <code>, which id indented properly, and converts it to user-friendly non-indented code.

"ngd-code" does prettify your code by removing;

  1. The first line feed
  2. The last line feed
  3. All html indentation, not code indentation, of all lines

Example

Without ngd-code 

                          foo
                            indent1
                              more indent
                            /indent1
                          /foo
With ngd-code 

                          foo
                            indent1
                              more indent
                            /indent1
                          /foo

ngd-image-drop Download Source

Enable to show image drag and drop from a file browser or a web browser
Whenever drops an image it, broadcasts an event, "ngd-image-dropped" with options, and options are;

  • {files: <<Files Dropped>>}
  • or, {url: <<URL of an image>>}
The event broadcasted are used by "ngd-image-preview" directive. For example; 

     scope.$on('ngd-image-dropped', function(event, options) {
      ....
     });
It also add a class when images are dragging. The class name is "dragover". The class, "dragover", will be removed when images are dropped.

ngd-image-preview Download Source

Captures "ngd-image-dropped" event, which is emitted by "ngd-image-dropped" directive or "ngd-image-paste" directive, and, add the image into the element.

There are two attributes to control how we display an image;

  • ngd-image-preview-width: width of an image when it is shown
  • ngd-image-preview-height: height of an image when it is shown

ngd-overlay Download Source

This works like a modal window with extra feature, which can be applied on the top of any element.

To show fully customized overlay element on;

  • full document as a modal window
  • or, on a specific element
The default css will be applied on background and contents as; 

/* For background */
{
  background:rgba(0,0,0,0.5);
}

/* For contents */
{
  background:#fff;
  display:inline-block;
  position:absolute;
  text-align: center;
}
NOTE:
To use this directive with `ng-include`, please remind that `ng-include` has its own scope. Since `ng-include` has isolated scope, any events, i.e. click, search for its own scope first. Thus, do not use variable directly for the event if intended scope is not ng-include itself. 
i.e. 
  NO:  ng-click="count = count +1" // increase count within ng-include scope
  YES: ng-clicl="increase(+1)      // call method increase by searching current scope and up.
To override this default css;
  • `!important` at the end of each attribute,
  • or, just simply apply `ngd-overlay-nocss`. With `*-nocss`, no inline css will be applied. 
    
        <div ngd-overlay="foo" ngd-overlay-nocss>Overlay</div>

ngd-viewport Download Source

This works like bootstrap "scrollspy" in AngularJS way

This directive, `ng-viewport`, watches scroll event of window and set the first element in the viewport to `NgViewport.currentEl`. Another directive `ng-viewport-nav` watches the changes of `NgViewprot.currentEl`, and set the corresponding element in the navigatior section

There are following directives that are used as attributes as part of `ngd-viewport`;

  • ngd-viewport

    • Collect all elements that matches the given selector, store it to "NgdViewpot.elements".
    • Watch the scroll of the element, and set the first element in the current viewport to "NgdViewport.currentEl".
    The value of this attribute is a css selector, and the element must have an id. The default value is 'a[id]'.

  • ngd-viewport-el

    Only used when `ngd-viewport` section is dynamically updated, i.e., `ng-repeat`. `ng-viewport` directive does not watch the changes of inner html. When inner httml of `ngd-viewport` is dynamically updated, you need to update it with `ngd-viewport-el` so that "NgdViewport.elements" are updated.

  • ngd-viewport-nav

    • Collect all elements that matches the given selector.
    • Watch the change of "NgdViewport.currentEl", and set the class name to the matching navigation element.
    The class name is "active" by default, but you can change it with `ngd-viewport-nav-class` attribute. The value of this directive is a css selector, and the selector must be attribute specific. The attribute value of the given selector is compared to the id of the viewport element selected, "NgdViewport.currentEl". Th default value is 'a[href]'

  • ngd-viewport-class

    Defines class name used by `ngd-viewport-nav`. Default, "active"

ngd-tree Download Source

Deep-level collapsale list with 'ngd-tree' attribute. This directive does;

  1. set the default style of ngd-tree if not set yet by adding the following css into the <head> tag 
    
      ul[ngd-tree] li { list-style: none; }
      ul[ngd-tree] li:not(.on) ul {  display: none; }
      ul[ngd-tree] li:before { content: " "; display: inline-block; width: 7%;
      ul[ngd-tree] li > * { display: inline-block; width: 90%; border-bottom: 1px solid #ddd}
      ul[ngd-tree] li.has-ul:before { content: "▸"; }
      ul[ngd-tree] li.has-ul.on:before { content: "▾"; }
  2. Find all <li> element that has &ul>, then set "has-ul" class name
  3. Register "click" event listener to collpase/uncollapse the list
