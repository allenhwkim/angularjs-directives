ngd-sidebar-below Download

Set floating sidebar below the header

Example

The good usage of this is the &asige> tag used in this page. From the following code, it always shows floating sidebar below the header tag 

  <aside ngd-sidebar-below="header">
When <header> is not visible it sets style to 
 position: fixed; top: 0
When <header> is visible it sets style to 
 position: absolute; top: null
