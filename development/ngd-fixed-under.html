ngd-fixed-under Download

Set fixed floating sidebar under the header

Example

The good usage of this is the &asige> tag used in this page. From the following code, it always shows floating sidebar under the header tag 

  <aside ngd-fixed-under="header">
When <header> is not visible it sets style to 
 position: fixed; top: 0
When <header> is visible it sets style to 
 position: absolute; top: null
