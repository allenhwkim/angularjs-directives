ngd-code Download

We want to type code and show to the users. However, <pre> tags cares every single space and does not care about your indentation for your html and ends up showing ugly. If you want to make it look pretty to the user, your html indentation must be ignored.

"ng-code" does read your html <code>, which id indented properly, and converts it to user-friendly non-indented code.

"ngd-code" does prettify your code by removing;

  1. The first line feed
  2. The last line feed
  3. All html indentation, not code indentation, of all lines

Example

Without ngd-code 

                          foo
                            indent1
                              more indent
                            /indent1
                          /foo
With ngd-code 

                          foo
                            indent1
                              more indent
                            /indent1
                          /foo
