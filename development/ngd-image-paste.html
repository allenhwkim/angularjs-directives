ngd-image-paste Download

Enable to paste image from a file browser.
Whenever paste an image, it broadcasts an event, "ngd-image-dropped" with options. The option is

  • {files: <<Files pasted>>}
The event broadcasted are used by "ngd-image-preview" directive. For example; 

     scope.$on('ngd-image-dropped', function(event, options) {
      ....
     });

NOTE: when copy/paste, it must be copied as an image not as a file, i.e. screenshot, or select from image editor.
